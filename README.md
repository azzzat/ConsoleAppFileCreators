# ConsoleAppFileCreators

1. В файле ConsoleAppFileCreator программа которая создает файл 
   со списком из 1000 единиц продукции.

2. В файле ConsoleAppFileReader программа которая считывает созданный выше файл 
   и дополнительно формирует набор файлов с тем же списком, но отсортированными по цене.
   
3. Все созданные файлы хранятся в папке Files папки репозитория.
